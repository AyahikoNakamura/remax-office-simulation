# RE/MAX オフィス経営シミュレーション

セミナー中に、オフィスオーナー候補へRE/MAXオフィス経営の収益構造を見せるためのシンプルなシミュレーターです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「300万」「500万」「1,000万」「3,000万」の利益フェーズを選びます。
3. 事業前提、ランク別フィー、オーナー取引額、ランク別採用計画を入力します。
4. 年間オフィス粗利、営業利益、目標利益との差額、年次シミュレーションをその場で確認します。
5. 「エージェントランク設定」で、登録ランクごとの月額Agフィー、還元率、達成後還元率を確認・編集します。
6. 「ランク別採用計画」で、各ランクの採用人数と1人あたり年間取引金額を設定します。
7. 「目標利益から逆算」で、必要年間オフィス粗利、必要取引額、Ag1人あたりの必要取引額を確認します。

## 現在の計算前提

- 年間オフィス粗利 = 仲介手数料のオフィス取り分 + Agフィー年間利益 + オーナー年間取引額の94%
- Agフィー年間利益 = エージェント数 × (月額Agフィー - 本部支払0.8万円) × 12か月
- 年間営業利益 = 年間オフィス粗利 - 月間固定費内訳合計 × 12か月
- 必要年間オフィス粗利 = 年間固定費 + 目標営業利益
- 必要Ag側取引額 = 必要年間オフィス粗利からAgフィー年間利益とオーナー年間取引額の94%を除いた不足粗利 ÷ オフィス取り分
- Ag1人あたり必要取引額 = 必要Ag側取引額 ÷ エージェント数
- 年間成約件数 = 年間取引額 ÷ 1件あたり平均仲介手数料
- ランク反映時のオフィス取り分 = (100% - RE/MAX支払分6%) × (100% - エージェント還元率)
- ランク別採用計画の取引由来オフィス粗利 = 採用人数 × ランク別の年間取引金額 × ランク別オフィス取り分

## セミナーでの見せ方

- まず利益300万円フェーズで黒字化に必要なラインを見せる
- 次に利益500万円、1,000万円、3,000万円へ進み、オフィスの成長段階を見せる
- 目標利益から逆算し、Ag採用人数ごとの必要取引額を見せる
- 最後に年次シミュレーションで、エージェント数が毎年3名純増し、生産性・固定費が一定割合で伸びた場合の利益推移を見せる

PDF「20260428_■Ag採用計画.pdf」の4パターンを参考値として反映しています。細かな数値は、実際のRE/MAX北海道の加盟条件、ロイヤリティ、研修費、広告費、オフィス賃料などに合わせて調整できます。
